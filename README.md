# ItBank

Frontend para TP desarollado con REACT + VITE

## Levanta el proyecto !

### 1. Clona the repository

```bash
git clone https://github.com/manuelKrivoy/ITBANK-2024.git
cd itbank
```

### 2. Instala las dependencias

```bash
npm install
```

### 3. Levanta el proyecto

```bash
npm run dev
```
